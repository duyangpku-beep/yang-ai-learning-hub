# Yang's AI Learning Hub

Personal brand site — bio, AI learning resources, projects, and contact.

## Stack

Pure HTML + CSS + Vanilla JS. No frameworks, no build tools.

- **Fonts**: EB Garamond (Google Fonts) + Microsoft YaHei fallback
- **Hosting**: Vercel (static)
- **Domain**: yangsailearninghub.com (planned)

## Local development

Just open `index.html` in a browser — no server required.

## Deployment

1. Push to GitHub
2. Import repo at vercel.com
3. Add custom domain in Vercel dashboard
4. Point DNS to Vercel nameservers
